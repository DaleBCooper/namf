# Particulate Matter App

Particulate Matter App is Android application for displaying data from sensors compatible with Sensor Community devices.
NAM is capable of sending data there.

Particulate Matter App (aka _Feinstaub App_) does not relay on SC infrastructure and requires
that device send data to its API. Mark this checkbox to start sending data.
